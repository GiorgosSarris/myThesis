# CognitoCode Usage Guide

## 1. Upload Code

- In **Step 1**, upload a `.zip` file with student source code.
- Click **Upload & Parse**.

## 2. Add Assignment Specs (Teacher Input)

- In **Title**, write the assignment title.
- In **Description**, paste the assignment statement/instructions.
- In **Quiz Requirements**, define quiz format and focus, for example:
  - number of questions
  - multiple-choice or true/false
  - concepts to emphasize
- Click **Apply Specifications**.

## 3. Generate Quiz

- In **Step 3**, choose **All Assignments** or **Selected Only**.
- The app shows:
  - how many assignments will be generated
  - a real-time estimated input token count for the selected assignments
- Click **Generate Quiz**.

## 4. View and Download

- In **View & Download**, pick an assignment.
- Click **View Aiken** to preview.
- Click **Download** to save the quiz text file.

## Notes
- Final token usage can differ depending on model tokenization and output length.
